GitView — GitHub Profile Finder
Project Overview

GitView is a simple and responsive web app that lets users search for any GitHub username and instantly view their public profile details and repositories.
It uses the GitHub REST API to fetch real-time data, showing profile info, repository list, and pagination for easy navigation.

Features

Search any GitHub username
View profile picture, followers, following, and public repo count
Display repositories with names and descriptions
Pagination (Next / Prev) for smooth browsing
Direct link to repositories (opens in new tab)
Clean and minimal user interface

 
Tech Stack

HTML – Structure of the web page
CSS – Styling and layout
JavaScript (ES6) – Functionality and API handling
GitHub REST API – Fetching user data and repositories
